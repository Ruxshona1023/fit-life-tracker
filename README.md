# FitLife Tracker 🏃‍♂️🔥

FitLife Tracker — bu sog'lom turmush tarzini kuzatib borish uchun zamonaviy veb-ilova. Ushbu loyiha orqali kunlik qadamlar sonini kuzatishingiz, sarflangan kaloriyalarni hisoblashingiz va haftalik faollik grafigini ko'rishingiz mumkin.

## Xususiyatlari
- **Zamonaviy Dizayn**: Dark mode va Glassmorphism stilidagi premium interfeys.
- **Qadamlar Kuzatuvchisi**: Kunlik qadamlar va masofani (km) real vaqtda hisoblash.
- **Kaloriya Hisoblagichi**: Faollik darajasiga qarab sarflangan kaloriyalarni avtomatik aniqlash.
- **Vizual Statistika**: Haftalik faoliyatni dinamik grafiklar orqali tahlil qilish (Chart.js).
- **Ma'lumotlar Saqlanishi**: Ma'lumotlar brauzer xotirasida (LocalStorage) saqlanadi va "Cloud Sync" simulyatsiyasiga ega.

## Texnologiyalar
- HTML5 & CSS3 (Vanilla)
- JavaScript (ES6+)
- [Lucide Icons](https://lucide.dev/)
- [Chart.js](https://www.chartjs.org/)
